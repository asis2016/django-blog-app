## Run Locally

Clone the project

```bash
  git clone https://github.com/asis2016/django-blog-app.git
```

Go to the project directory

```bash
  cd django-blog-app
```

Start the project

```bash
  docker build .
```

```bash
  docker-compose up -d
```
